💻 클로드 코드(Claude Code)란?

클로드가 직접 내 컴퓨터 파일을 열고, 읽고, 수정하고, 저장까지 해주는 코딩 전문 도구예요.

❌ 일반 클로드

  • 파일을 못 봐요
  • 내가 코드를 복사해서 붙여넣어야 해요
  • 실제 파일에 손을 못 대요
  • 말로만 알려줘요

✅ 클로드 코드

  • 파일을 직접 열어요
  • 폴더 전체를 뒤져서 읽어요
  • 코드를 고치고 저장까지
  • 실제로 명령어도 실행해요

부엌에 직접 오는 요리사

일반 클로드가 "전화로 레시피 알려주는 요리사"라면,
클로드 코드는 "우리 집 부엌에 직접 와서 요리까지 해주는 요리사"예요.

🔑 핵심: 말로만 알려주는 게 아니라 직접 내 파일을 만져서 결과물을 만들어줘요.

🖥️ 코워크(Cowork)란?

🚗

자동 기어 자동차

클로드 데스크톱 앱 안에 있는 기능이에요.
어렵게 터미널(검은 화면)을 몰라도 클릭 몇 번으로 클로드 코드를 쓸 수 있어요.

🚗 비유: 수동 기어(터미널)는 배우기 어렵지만, 코워크는 자동 기어 + 내비 + 주차 보조까지 다 달린 차예요. 초보도 바로 운전 가능!

🖥️

데스크톱 앱 (코워크 포함) 👍 초보 추천

컴퓨터에 앱을 설치해서 사용. 코워크 기능 포함!
클릭 몇 번으로 문서·파일·자동화 작업 가능. 개발 지식 0% OK.

카카오톡을 설치하면 쭉 쓸 수 있잖아요? 데스크톱 앱도 그래요.
한번 설치하면 바탕화면에서 바로 열 수 있고, 아이콘 클릭 → 바로 사용이에요.
🧩

VS Code 📚 코딩 배우는 분

코딩 전용 프로그램(VS Code) 안에서 클로드 코드 사용.
코드를 작성하면서 바로 옆에서 클로드에게 물어볼 수 있어요.
📝 워드 문서 쓰면서 맞춤법 검사기 쓰는 것 같아요!
워드에서 글 쓸 때 맞춤법 검사기가 밑줄 그어주잖아요?
VS Code에서 코드 쓸 때 클로드가 바로 옆에서 실시간으로 도와줘요.
🔷

JetBrains 💼 전문 개발자

IntelliJ, PyCharm, WebStorm 같은 전문 개발 도구 안에서 사용.
대형 회사 개발자들이 주로 쓰는 고급 환경이에요.
🏥 의사 전용 의료 장비 같아요!
병원 의사들이 일반 체온계 대신 정밀 의료기기를 쓰듯이,
전문 개발자들이 더 정밀한 개발 도구(JetBrains) 안에서 클로드를 써요.
⌨️

터미널 (CLI) 🚀 고급자용

검은 화면에 명령어를 직접 입력해서 사용. 가장 강력하고 빠름.
마우스 대신 키보드만으로 모든 걸 제어해요.
🍳 재료부터 직접 요리하는 셰프 같아요!
배달앱으로 시키면 쉽지만, 직접 요리하면 원하는 대로 다 만들 수 있잖아요?
터미널도 어렵지만 뭐든 마음대로 제어할 수 있는 가장 강력한 방법이에요.
⚠️ 처음엔 어렵게 느껴져도 괜찮아요. 코워크나 웹으로 먼저 시작해보세요!

✨ 뭘 할 수 있나요?

공식 문서 기준 8가지 기능이에요. 바로 써볼 수 있는 실제 프롬프트도 같이 넣었어요!

🧹

귀찮은 작업 자동화

반복적이고 지루한 일을 대신 해줘요. 테스트 작성, 오류 수정, 릴리스 노트처럼 시간 잡아먹는 일들이요.
💬 이렇게 입력해봐요!
이 프로젝트의 모든 함수에 주석 달아줘
변경 사항 보고 릴리스 노트 초안 써줘
중복 코드 찾아서 정리해줘
✅ 결과: 클로드가 파일을 직접 열고 수정한 다음 "이렇게 바꿨어요"라고 보고해줘요.
🔨

기능 만들기 & 버그 수정

원하는 걸 말로 설명하면 코드를 짜줘요. 에러 메시지를 그냥 붙여넣으면 직접 찾아서 고쳐줘요.
💬 이렇게 입력해봐요!
로그인 폼에서 이메일 형식 검사 기능 추가해줘
TypeError: Cannot read property 'map' of undefined ← 이 에러 고쳐줘
버튼 누르면 로딩 스피너 보이게 만들어줘
✅ 결과: 코드를 고치기 전에 먼저 "이렇게 바꿀게요, 괜찮아요?" 하고 확인을 받아요.
📝

Git 커밋 & PR 자동화

코드 변경 저장(커밋), 협업 요청(PR) 생성을 말로 시킬 수 있어요. Git 명령어 몰라도 완전 OK!
💬 이렇게 입력해봐요!
변경 사항 적절한 메시지로 커밋해줘
지금까지 변경한 게 뭔지 요약해줘
PR 설명 초안 써줘
✅ 결과: 클로드가 바뀐 파일 목록을 보고 커밋 메시지까지 알아서 작성해줘요.
🔌

외부 도구 연결 (MCP)

구글 드라이브, Slack, Jira, GitHub 등 외부 앱과 연결해서 데이터를 가져오거나 업데이트해요.
💬 MCP 연결 후 이렇게!
Slack #general 채널에 배포 완료 메시지 보내줘
Jira에서 내 담당 티켓 목록 가져와줘
구글 드라이브 '회의록' 폴더에서 오늘 파일 열어줘
✅ 결과: 클로드가 앱들 사이를 오가며 대신 처리해줘요. (MCP 설정 먼저 필요)
⚙️

나만의 규칙 & 워크플로우

CLAUDE.md로 규칙 정해두기, Skills로 자주 쓰는 명령어 저장, Hooks로 자동 실행 설정.
💬 이렇게 입력해봐요!
CLAUDE.md 파일 만들고 "항상 한국어로 답해줘" 규칙 추가해줘
/deploy 명령어 만들어줘 — 테스트 후 자동 배포하는 것
파일 저장할 때마다 자동으로 린트 실행하도록 Hook 설정해줘
✅ 결과: 한 번 설정하면 매번 말 안 해도 클로드가 기억하고 자동으로 해줘요.
👥

에이전트 팀 실행

여러 클로드가 동시에 각자 맡은 부분을 처리해요. 큰 작업도 훨씬 빠르게 완성!
💬 이렇게 입력해봐요!
로그인·상품목록·결제·디자인 기능을 동시에 개발해줘
모든 페이지를 병렬로 테스트해줘
여러 에이전트로 나눠서 DB 마이그레이션 처리해줘
✅ 결과: 나루토 분신술처럼 클로드가 여럿으로 나뉘어 동시에 일해요. ⚡
⏰

반복 작업 예약 (스케줄)

매일 아침 PR 검토, 매주 코드 점검 등 정기 작업을 자동 예약해서 실행해요.
💬 이렇게 입력해봐요!
매일 오전 9시에 미완성 PR 목록 정리해줘
매주 월요일에 지난주 커밋 요약 보고서 써줘
30분마다 API 상태 확인하고 오류 있으면 알려줘
✅ 결과: 알람 맞춰두면 자동으로 실행돼요. 코워크 스케줄 기능으로 쉽게 설정 가능!

어디서나 작업

GitHub 이슈 #42 해결해줘 (폰으로 확인하면서 지시)

🚀 처음 시작하는 법

터미널로 시작하는 방법이에요. 코워크(데스크톱 앱)를 쓰면 앱 실행 후 로그인만 하면 돼요!

1

⬇️ 설치하기

Mac/Linux: curl -fsSL https://claude.ai/install.sh | bash

Windows PowerShell: irm https://claude.ai/install.ps1 | iex

Homebrew: brew install --cask claude-code

2

🔑 로그인하기

터미널에서 claude를 입력하면 로그인 화면이 나와요.

Pro, Max, Teams, Enterprise 구독 또는 Console(API) 계정이 필요해요.

한번 로그인하면 다음부터 자동 연결!

3

📂 프로젝트 폴더로 이동

cd 내폴더경로 → claude 입력

환영 화면이 나오면 성공! 이전 대화를 이어하려면 /resume을 입력하세요.

4

❓ 첫 번째 질문 해보기

이 프로젝트는 무엇을 하나요? 라고 입력해보세요.

클로드가 알아서 파일들을 분석하고 설명해줘요!

5

✏️ 코드 변경 해보기

주 파일에 hello world 함수 추가해줘

클로드가 뭘 바꿀지 먼저 보여주고 승인을 물어봐요. 파일 수정 전 항상 허락 받음!

6

💾 Git으로 저장하기

변경 사항을 설명이 담긴 메시지로 커밋해줘

Git을 몰라도 말로 시키면 클로드가 알아서 해요!

7

🐛 버그 수정 해보기

사용자가 빈 양식을 제출할 수 있는 버그가 있어요, 수정해줘

에러 메시지가 있으면 그냥 붙여넣으면 돼요!

8

🎯 다른 것도 해보기

계산기 함수에 대한 단위 테스트 작성해줘

내 변경 사항을 검토하고 개선 사항 제안해줘

자주 쓰는 명령어

명령어하는 일
claude클로드 코드 시작
claude "할 일"한 번만 시키기 (예: claude "빌드 오류 고쳐줘")
claude -p "질문"답 받고 바로 끝내기
claude -c이전 대화 이어하기
claude commitGit 커밋 만들기
/clear대화 기록 지우기
/help명령어 목록 보기
/login계정 로그인 또는 전환
/resume이전 대화 이어서 하기
exit / Ctrl+C종료하기
💡 팁: "버그 수정"보다 "로그인할 때 비밀번호 틀려도 아무 반응 없는 버그 수정해줘"처럼 구체적으로 말할수록 더 잘 해줘요!

카카오톡 꾸미기 기호와 같아요

카카오톡에서 **굵게**라고 쓰면 굵게로 보이는 것처럼, .md도 특수기호로 글 서식을 만들어요.

💡 왜 쓸까요? CLAUDE.md, README.md, SKILL.md… 클로드 코드에서 나오는 대부분의 메모 파일이 이 형식이에요. 용량이 작고 GitHub에서 예쁘게 보여줘요.

📋 CLAUDE.md — 클로드의 메모장

CLAUDE.md는 클로드가 매 대화를 시작할 때 자동으로 읽는 필독 파일이에요.

🐟

하지만 지갑에 메모를 붙여두면 매번 꺼내서 읽을 수 있어요!

클로드도 마찬가지! 대화를 끝내면 클로드는 기억을 잃어요. 하지만 CLAUDE.md에 규칙을 적어두면 다음 대화에서 그 파일을 읽어서 기억해요.

자동 메모리 — 클로드가 스스로 메모!

작업하면서 배운 것을 스스로 CLAUDE.md에 기록해요. 내가 따로 안 알려줘도 돼요!

💼 비유: 새 직장에서 자주 쓰는 비밀번호, 서버 주소를 스스로 수첩에 적어두는 신입 직원 같아요. (/memory로 확인 가능)

⏱️ 세션과 컨텍스트란?

📞

세션 = 전화 통화 1회

친구한테 전화해서 30분 수다 떨고 끊으면 = 세션 1개.
다음 날 다시 전화하면 새 세션. 어제 통화 내용은 기억 못 해요.

CLAUDE.md에 적어두면 새 세션에서도 기억해요!

🧠 컨텍스트 윈도우 — 메모지 1장

대화가 길어질수록 메모지가 채워져요. 꽉 차면 오래된 내용부터 사라져요.

  🧠 현재 메모리 사용량 (약 72%)
🆕 대화 시작⚠️ 꽉 참 → 오래된 내용 삭제
초록 — 여유 있음. 클로드가 잘 기억해요.
노랑 — 조금 꽉 차가요. /clear로 정리하면 좋아요.
빨강 — 위험! 초반 내용을 잊기 시작해요. 새 세션 시작하세요.
💡 팁: 세션이 너무 길면 클로드가 느려지거나 초반 내용을 잊어요. CLAUDE.md에 중요 규칙을 적어두면 새 세션에서도 기억해요!

🔌 MCP — 외부 도구 연결

MCP (Model Context Protocol)는 클로드가 외부 앱·서비스와 연결하기 위한 개방형 표준이에요.

MCP 연결 그림
📁 구글 드라이브
💬 Slack
📋 Jira
🐱 GitHub
📔 Notion
🔧 내 커스텀 도구
⬇ MCP 케이블 ⬇
🤖 클로드 코드

연결 후: "구글 드라이브에서 오늘 공유된 문서 요약해줘" 같은 것도 가능!

📺

HDMI 케이블 같은 것

노트북을 TV에 연결하려면 HDMI 케이블이 필요하듯이,
클로드를 Slack, GitHub에 연결하려면 MCP가 필요해요.

💡 설치 방법: 코워크(데스크톱 앱) 설정 화면에서 MCP 서버 추가 가능. Slack, GitHub, Google Drive 같은 인기 MCP들이 마켓플레이스에 있어요.
MCP 없이MCP 연결 후

MCP를 연결하면 클로드가 할 수 있는 일이 훨씬 많아져요

📚 전체 용어 정리

클로드 코드에서 만나는 모든 용어를 한눈에!

용어한국 이름한 줄 설명
Claude클로드Anthropic이 만든 AI 어시스턴트. 챗GPT 같은 것.
Claude Code클로드 코드클로드가 실제 파일을 열고 수정하는 코딩 전문 도구.
Cowork코워크데스크톱 앱에서 비개발자도 쓸 수 있는 GUI 기능.
.md / Markdown마크다운#으로 제목, **로 굵기 표시하는 메모장 파일 형식.
CLAUDE.md클로드 메모장클로드가 매 세션마다 읽는 필독 규칙 파일. 도리의 메모지.
Session세션클로드와 나누는 한 번의 대화. 전화통화 1회.
Context Window컨텍스트한 세션에서 클로드가 기억할 수 있는 최대 양. 메모지 1장.
Auto-Memory자동 메모리클로드가 작업하며 배운 것을 스스로 CLAUDE.md에 기록.
/슬래시 명령어단축 명령어클로드 코드 안의 단축키. /help, /clear 등.
MCP외부 도구 연결Model Context Protocol. 클로드를 외부 서비스에 연결하는 HDMI 케이블.
Hooks자동 실행 트리거클로드 작업 전후에 자동으로 실행되는 명령어. 스마트폰 자동화 앱 같은 것.
Skills나만의 명령어 패키지복잡한 작업을 /명령어로 저장해두는 레시피 카드.
Sub-agents서브에이전트여러 클로드가 동시에 작업. 나루토 분신술.
Git버전 관리코드 변경 기록 시스템. 게임 세이브 포인트.
Commit커밋코드 변경 내용을 Git에 저장. 세이브 포인트 1개.
Pull Request풀 리퀘스트변경 코드를 검토받고 합쳐달라는 요청.
Terminal / CLI터미널텍스트로 컴퓨터에 명령하는 검은 화면.
